📖 Como usar o Simulador MRUV

Este simulador permite explorar de forma visual e interativa o Movimento Retilíneo Uniformemente Variado (MRUV) — o movimento no qual a aceleração é constante e a velocidade varia linearmente com o tempo. Siga as instruções abaixo para tirar o máximo proveito da ferramenta.

🎚️

1. Configure os Parâmetros

Use os dois sliders no painel da esquerda:

  • Velocidade Inicial (v₀): velocidade com que o carro começa a se mover. Varia de 0 a 30 m/s.
  • Aceleração (a): taxa de variação da velocidade. Valores negativos fazem o carro desacelerar. Varia de −5 a +10 m/s².
▶️

2. Inicie e Observe

Clique em INICIAR para começar a simulação. Observe:

  • O carro mover-se na pista
  • As setas (vetores) crescerem ou encolherem em tempo real
  • Os gráficos serem desenhados dinamicamente
  • Os valores de v, a, t e posição atualizarem nos cartões superiores
🏹

3. Leia os Vetores

  • Seta verde (Velocidade): quanto maior, mais rápido o carro. Ela cresce ao longo do tempo quando há aceleração positiva.
  • Seta amarela (Aceleração): tamanho fixo — representa que a aceleração é constante no MRUV. Se estiver para a esquerda, o carro está a travar.
📡

4. Use o Radar de Velocidade

Defina o limite do radar com o slider vermelho. Se o carro passar pelo poste do radar com velocidade acima do limite, um flash dispara e um ticket é exibido com os dados do momento exato (v, t, s). Use para aplicar a Equação de Torricelli!

🐢

5. Câmera Lenta (Slow Motion)

Use os botões 0.25×, 0.5×, 1× ou 2× para controlar a velocidade da simulação. Em câmera lenta, é possível observar passo a passo como a aceleração altera a velocidade a cada instante.

📊

6. Analise os Gráficos

  • Gráfico v(t): deve ser uma reta. A inclinação da reta é exatamente a aceleração.
  • Gráfico s(t): é uma parábola quando há aceleração (curva). Passe o mouse sobre os pontos para ver os valores exatos.

Equações do MRUV — Referência Rápida

v(t) = v₀ + a·tFunção horária da velocidade
s(t) = s₀ + v₀·t + ½·a·t²Função horária da posição
v² = v₀² + 2·a·ΔsEquação de Torricelli (sem tempo)
